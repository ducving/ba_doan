# 🔧 Fix Lỗi 404 Not Found

## Vấn đề
Khi test trên Postman, nhận được lỗi **404 Not Found** khi gọi API.

## ✅ Giải pháp

### Cách 1: Truy cập trực tiếp file PHP (Khuyến nghị)

Thay vì dùng router, truy cập trực tiếp file PHP:

**URL mới:**
- Register: `http://localhost/doan/api/register.php`
- Login: `http://localhost/doan/api/login.php`
- Verify Token: `http://localhost/doan/api/verify_token.php`

**Lưu ý:** Đảm bảo đường dẫn đúng. Nếu project của bạn ở thư mục khác, hãy điều chỉnh URL.

---

### Cách 2: Kiểm tra Apache mod_rewrite

1. **Kiểm tra mod_rewrite đã bật chưa:**
   - Mở `httpd.conf` trong XAMPP/WAMP
   - Tìm dòng: `#LoadModule rewrite_module modules/mod_rewrite.so`
   - Bỏ dấu `#` để bật: `LoadModule rewrite_module modules/mod_rewrite.so`
   - Restart Apache

2. **Kiểm tra AllowOverride:**
   - Trong `httpd.conf`, tìm section `<Directory>`
   - Đảm bảo có: `AllowOverride All`
   - Restart Apache

---

### Cách 3: Kiểm tra đường dẫn project

**Xác định đường dẫn chính xác:**
1. Tạo file `test.php` trong thư mục `doan/`:
   ```php
   <?php echo "OK"; ?>
   ```
2. Truy cập: `http://localhost/doan/test.php`
3. Nếu thấy "OK" → đường dẫn đúng
4. Nếu không → kiểm tra lại đường dẫn trong XAMPP/WAMP

**Các đường dẫn phổ biến:**
- XAMPP: `C:\xampp\htdocs\doan\`
- WAMP: `C:\wamp64\www\doan\`
- Laragon: `C:\laragon\www\doan\`

---

### Cách 4: Sửa URL trong Postman

Nếu project của bạn ở đường dẫn khác, hãy sửa URL:

**Ví dụ:**
- Nếu ở `http://localhost/` (root): `http://localhost/api/register.php`
- Nếu ở `http://localhost/myproject/`: `http://localhost/myproject/api/register.php`

---

## 🧪 Test nhanh

### Test 1: Truy cập trực tiếp file
Mở trình duyệt, truy cập:
```
http://localhost/doan/api/register.php
```

**Kỳ vọng:**
- Nếu thấy JSON response (có thể là lỗi validation) → File hoạt động ✅
- Nếu thấy 404 → Vấn đề về đường dẫn ❌

### Test 2: Test với Postman
1. Method: `POST`
2. URL: `http://localhost/doan/api/register.php`
3. Headers: `Content-Type: application/json`
4. Body:
   ```json
   {
       "name": "Test",
       "email": "test@test.com",
       "password": "Password123"
   }
   ```
